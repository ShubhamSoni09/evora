# evora

AI dementia companion for patients, caregivers, and family — voice calls, emotional support, and clinical oversight.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). See [SETUP.md](./SETUP.md) for API keys (xAI, ElevenLabs, Twilio, Inngest).

## Deploy

Push to GitHub and import into [Vercel](https://vercel.com). Add environment variables from SETUP.md.
